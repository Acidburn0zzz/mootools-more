Class: Form.Validator {#Form-Validator}
=====================================

Contains the messages for [Form.Validator][] in Ukrainian.

### Authors

### Usage

Simply change the language property to *uk-UA*.

	Locale.setCurrent("uk-UA");

### Note

Only encoding for Ukrainian is *utf-8*
